# fog-offloading-simulation
# Fog Computing Task Offloading Simulation

## Overview

This repository contains the complete Java implementation of three task offloading algorithms for fog computing:

- **Q-learning** (Reinforcement Learning)
- **Min-Min** (Heuristic)
- **Load-Based** (Shortest Queue)

The algorithms are evaluated on latency, energy consumption, and network usage across IoT device densities from 100 to 1,500 devices.
